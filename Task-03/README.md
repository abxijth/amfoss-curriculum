# Task 03: Wireframe

## Overview

- Making **low-fidelity** wireframes of the **MeloFi** application that display its structure and layout on both web and mobile platforms is the main goal of this task.
Before beginning detailed UI design, the wireframes specify the positioning of important interface components, navigation flow, and user interaction points.

## Goal

- To use **wireframes** to clearly illustrate **MeloFi's** overall user flow, screen hierarchy, and layout structure, laying the groundwork for upcoming design and development phases.
